# bigpun.dev

Personal website built with TanStack Router and Vite.

## Development

```sh
pnpm install
pnpm dev
```

## Tech Stack

- [TanStack Router](https://tanstack.com/router)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
